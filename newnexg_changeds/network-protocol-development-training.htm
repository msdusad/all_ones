Advanced Diploma in Network Protocol Development (ADNPD)

This training is designed to equip the candidate with the development skills and networking concepts with the hands on experience on real-time telecom network protocol testing.

Who Should Do ?

Bachelor Of Technology, Bachelor Of Engineering, Bachelor Of Science, Bachelor Of Computer Application, Master Of Technology, Master Of Engineering, Master Of Science, Master Of Computer Application.


Modules Covered in the Course

Basics Of Networking

  • OSI/TCP-IP layers
  • Networking Topologies
  • Networking Devices
  • their functions & practical uses
  • Data flow models, Protocols & Standards
  • IP addressing & sub-netting
  • Ethernet cabling
  • Next Generation Wireless Network
    •

    L2 Protocols and Technologies

  • Switching Architecture
  • ARP
  • Vlan
  • STP
  • RSTP
  • MSTP
  • 1Q, QinQ
    •

    L3 Protocols and Technologies

  • Router Architecture
  • IP headers & working
  • ICMP
  • RIP V1
  • RIP V2
  • OSPF
  • BGP
  • IGRP
  • EIGRP
  • Redistribution
    •

    Application and Transport Layer Protocols

  • HTTP
  • FTP
  • Telnet
  • DNS
  • DHCP
  • TCP
  • UDP
    •

    Protocol Development Using C & C++

  • Basic Programming
  • Unix System Programming & Data Structure Programming
  • Project - Protocol Development
